# LovePulse v2.0

LovePulse — это современное веб-приложение (PWA) для отслеживания и анализа твоих романтических связей, дружеских отношений и семейных контактов.

## ✨ Что нового в v2.0

- **Полный редизайн интерфейса** — современный минималистичный дизайн с цветовой палитрой на основе мятного зелёного (#4cc9a0)
- **Система аутентификации** — регистрация и авторизация с изоляцией данных каждого пользователя
- **Единый HTML-файл** — всё приложение в одном файле, не требует сборки или сервера
- **Новая архитектура навигации** — 5 основных вкладок (Лента, Связи, Статистика, Дневник, Профиль)
- **Health Score** — алгоритм расчёта "здоровья" каждой связи на основе активности
- **LovePlus+** — система преимуществ для расширенного функционала

## 🎯 Функциональность

### Основное
- 📱 **Лента** — краткий обзор активности, важные события и напоминания
- 👥 **Связи** — управление контактами с категоризацией (Партнёр, Друг, Семья, Коллега, Бывший)
- 📊 **Статистика** — графики активности, топ людей, анализ паттернов общения
- 📓 **Дневник** — запись моментов с автоматическим тегированием людей
- 👤 **Профиль** — настройки, импорт/экспорт данных, управление аккаунтом

### Особенности
- **Мультиюзерность** — каждый пользователь имеет свой аккаунт и изолированные данные
- **Темная/светлая тема** — автоматическая подстройка под систему или ручной выбор
- **Оффлайн-работа** — Service Worker кэширует всё необходимое для работы без интернета
- **Экспорт/импорт** — резервное копирование данных в JSON-формате
- **Быстрая установка** — установка на главный экран мобильного устройства одной кнопкой

## 🚀 Запуск

### Локально
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/dantemorrigan/lovepulse.git
   cd lovepulse
   ```

2. Запустите локальный сервер:
   ```sh
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (http-server)
   npx http-server
   ```

3. Откройте в браузере:
   ```
   http://localhost:8000
   ```

### На мобильном
1. Откройте приложение в браузере мобильного телефона
2. Нажмите кнопку меню браузера и выберите "Добавить на главный экран"
3. Приложение установится как полноценное PWA

## 📝 Первый запуск

1. **Регистрация** — создайте аккаунт (email + пароль)
2. **Онбординг** — пройдите 4 слайда с инструкциями
3. **Начинайте** — добавляйте людей и записывайте моменты

## 🎨 Технические детали

### Стек
- **Язык** — Vanilla JavaScript (ES6+)
- **Хранилище** — localStorage (per-user namespacing)
- **Шрифты** — Fraunces (заголовки) + Geist (текст)
- **Цветовая схема** — CSS-переменные, поддержка светлой/тёмной темы

### Архитектура
- **Одностраничное приложение** (SPA) с 5 представлениями
- **Модульная система функций** (по категориям: auth, routing, state management, etc.)
- **Реактивное обновление** — изменения в state → перерендеринг UI

### Безопасность
- Пароли хешируются SHA-256 (+ fallback на djb2)
- Данные хранятся локально, никогда не отправляются на сервер
- Каждый пользователь изолирован (свой ключ в localStorage)

## 🔒 Конфиденциальность

LovePulse хранит ВСЕ данные локально на вашем устройстве:
- Никакие сведения не отправляются на серверы
- Данные остаются видны только вам
- При удалении браузера данные удаляются (если не сделали резервную копию)
- Экспортируйте важные данные через функцию экспорта в Профиле
