# LovePulse

PWA-дневник отношений и трекер связей. Храни всё локально — никакого сервера, никакой регистрации.

**Открыть:** https://dantemorrigan.github.io/lovepulse/

---

## Что умеет

- **Связи** — добавляй людей с ролью, статусом, заметкой, важной датой и напоминанием
- **Дневник** — пиши записи, тегай людей, отмечай настроение
- **Health Score** — автоматический «пульс» каждой связи на основе частоты записей
- **Статистика** — графики активности, топ упоминаний, карта связей, mood-аналитика
- **Годовщины** — карточка с ближайшими важными датами на главном экране
- **Поиск** — глобальный поиск по людям и записям
- **Экспорт** — JSON (резервная копия) и PDF (для печати)
- **Тёмная тема** — переключается в профиле
- **Оффлайн** — Service Worker кэширует приложение

## Стек

| | |
|---|---|
| Frontend | Vanilla JS (ES6+), HTML, CSS — всё в одном `index.html` |
| Хранение | `localStorage` — данные только на устройстве |
| Хостинг | GitHub Pages |
| Оффлайн | Service Worker |
| Установка | PWA — устанавливается на главный экран |

## Файлы

```
index.html          — приложение (HTML + CSS + JS)
manifest.json       — PWA-манифест
sw.js               — Service Worker
icons/              — иконки приложения
screenshots/        — скриншоты для PWA store
```
