# LovePulse

Веб-приложение (PWA) для отслеживания романтических и личных связей — с облачной синхронизацией и мультиустройственностью.

**Доступно онлайн:** https://dantemorrigan.github.io/lovepulse/

## Функциональность

- **Лента** — краткий обзор активности, важные события
- **Связи** — управление контактами с категоризацией (Партнёр, Друг, Семья, Коллега, Бывшие)
- **Статистика** — графики активности, топ людей, анализ паттернов общения
- **Дневник** — запись моментов с автоматическим тегированием людей
- **Профиль** — настройки, импорт/экспорт данных, управление аккаунтом

### Особенности

- **Аутентификация** — регистрация и вход через Supabase Auth
- **Облачная синхронизация** — данные доступны на всех устройствах
- **Оффлайн-работа** — Service Worker кэширует приложение для работы без интернета
- **Тёмная/светлая тема** — ручной выбор или автоматически по системе
- **PWA** — устанавливается на главный экран мобильного устройства

## Стек

- **Frontend** — Vanilla JavaScript (ES6+), HTML, CSS
- **Auth & Backend** — Supabase (Auth + PostgreSQL)
- **Hosting** — GitHub Pages
- **Офлайн** — Service Worker

## Структура файлов

| Файл | Назначение |
|------|-----------|
| `index.html` | Единственный HTML-файл, содержит HTML + CSS + JS |
| `manifest.json` | PWA манифест (имя, иконки, цвета) |
| `sw.js` | Service Worker для кэширования и оффлайн-режима |
| `supabase/` | Миграции базы данных |
